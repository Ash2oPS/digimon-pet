# Sprite Slots

Sprites are not bundled with the scaffold.

Expected layout:

```text
assets/sprites/<species_id>/idle.png
assets/sprites/<species_id>/walk.png
assets/sprites/<species_id>/sleep.png
assets/sprites/<species_id>/eat.png
assets/sprites/<species_id>/train.png
```

Use PNG spritesheets. If a file is missing, the app renders a placeholder.
